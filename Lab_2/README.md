# ELTEX Lab №2


## Usage:

Compile Lab №2:
```makefile
make Lab_2    
```
Run Lab №2:     
```makefile
make run     
```
 Remove executables files:
```makefile
make clear  
```
 Compile generate programme:
```makefile
make gen    
```
 Create test file:
```makefile
make example 
```
Check time for work:
```makefile
make ctest  
```
 Compile old version Lab №2 and check work time for example file:
```makefile
make old    
```
Make code in one style: 
```makefile
make indent  
```
